# **App Name**: Geek Haven

## Core Features:

- Product Catalog: Display a wide variety of video games, consoles, and geek merchandise.
- Search and Filter: Allow users to search products by name and filter by category, platform, price, and other relevant attributes.
- User Authentication: Enable user registration, login, and account management for personalized experience and order tracking.
- Shopping Cart: Allow users to add products to a cart, view cart contents, and proceed to checkout.
- Order Processing: Manage orders, payments, and shipping information securely.
- Recommendations: Suggest other products to the user based on what is currently in the cart.
- Localization: Offer the website in Basque/Euskera language.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to evoke a sense of sophistication and reliability, reflecting the vast collection of products.
- Background color: Light gray (#F2F2F2), a very low saturation version of the primary color, creating a clean and neutral backdrop that allows the products to stand out.
- Accent color: Electric purple (#826AED), an analogous hue which is both vibrant and distinctive.
- Font: 'Inter', a sans-serif font, will be used for both headlines and body text, because of its modern and neutral aesthetic.
- Use flat, modern icons for navigation and product categories.
- Employ a grid-based layout with clear sections for products, search, and user information.
- Subtle animations for product transitions and user interactions to enhance the browsing experience.
